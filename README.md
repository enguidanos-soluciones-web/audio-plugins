## Audio Plugins

### 1. Take My Headphones (Crossfeed)

Functional emulation of the Matrix section of the SPL Phonitor 3 and more.

![screenshot](./docs/take_my_headphones.png)

### Releases

```bash
# patch: 0.1.0 → 0.1.1
cargo release patch -p take-my-headphones

# minor: 0.1.0 → 0.2.0
cargo release minor -p take-my-headphones

# major: 0.1.0 → 1.0.0
cargo release major -p take-my-headphones
```
